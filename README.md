# Bingo
Simple Bingo App.

## Requirements
- HTML 5
- CSS 3
- ECMAScript 6

## Specifications
- The range is from 1 to 75 in default.
- The current number appears on left side, and histories are appear on right side.
- Just click the button to step!

## Installation
```bash
git clone https://github.com/Siketyan/Bingo.git
```
